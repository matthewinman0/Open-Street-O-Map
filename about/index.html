About Open Street O-Map


What Is OSOM?

OSOM or Open street orienteering map, is a worldwide orienteering map with contours and basic vegitiation included. It uses styles to establish that true classic orienteering look with osm data from OpenFreeMap.

Contours and hillshade data come from Mapterhorn and the Terrarium AWS S3 Bucket with contours converted from the data using maplibre-contour with Maplibre GL JS to render the vector tiles and output to the user.

The project has tools that let users upload their own maps to layer them ontop. This then means that the project can display maps in 3d as it has terrain maps built in, allowing for new geeking visualisations.


Development

I found mapant really cool in countries like Sweden and Norway, but the UK doesn't have mapant. so I attempeted to make my own.

After this failed I thought of Open Street map, and started to think of a project with intentions to make a decent map of the whole world to ISOM spec using open source, with no tiles hosted on my end. (although this soon may have to change)

I have experianced many challanges with georeferancing, programming and website design as this is my first solo javascript project. I iterated early on with Ai helping to set up a demo OSOM, then have slowly reprogrammed and polished without Ai into what it is today.

I'm hoping to improve OSOM with barriers (Walls, fences, etc) and correct georeferancing for import maps
